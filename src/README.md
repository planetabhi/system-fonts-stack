# System Fonts Stack
SCSS, CSS system fonts stack.

### Install
Install from your terminal via npm.

```
npm i system-fonts-stack
```

### Usage
```scss
// Place this at the top of your SCSS/CSS file
@import "system-fonts-stack";
```
